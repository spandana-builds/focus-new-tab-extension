## Calm Focus New Tab 

A minimal Chrome New Tab extension that replaces the default new tab with a calm, distraction-free dashboard for focus and clarity.

## Features
- Live time and date
- Daily motivational quote
- One-task focus input (saved using LocalStorage)
- Clean, calming UI

## Tech Stack
- HTML
- CSS
- JavaScript
- Chrome Extensions (Manifest V3)

## How to Install (Local)
1. Clone or download this repository
2. Open Chrome and go to `chrome://extensions`
3. Enable **Developer mode**
4. Click **Load unpacked**
5. Select the project folder
6. Open a new tab 

## What I Learned
- Chrome Extension architecture (Manifest V3)
- Content Security Policy (CSP) restrictions
- Event handling using JavaScript listeners
- Persistent storage with LocalStorage
